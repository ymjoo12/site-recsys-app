# site-recsys-app

### app (프론트 엔드)
- React Native 를 이용하여 android, ios 동시 개발
```
프레임 워크: React Native (Typescript)
개발 툴: expo
패키지 관리: yarn
상태 관리: MobX
```

### backend (백엔드)
- Django 를 이용하여 DB 관리 및 api 개발
- 추후 필요한 기능들을 Django app 으로 모듈화 하여 추가
```
프레임 워크: Django (Python)
DB: Aamazon RDS MySQL
서버: Amamzon ECS
```
